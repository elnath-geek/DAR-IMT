# DAR-IMT

The repository for my system DAR-IMT.

# Getting Started

### Web

Requirement: Node v18.14.2

```
cd web
npm ci
npm run build
npm start
```

Please open the link below  
http://localhost:3000?user=test_user

### Server

```
cd server/src
pip install pipenv
python -m pipenv install
python -m pipenv run python app.py
```
